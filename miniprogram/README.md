## 📚 文件资源目录

```text
├─ config                 # 打包配置
├─ src
│  ├─ api                 # api 接口请求
│  ├─ assets              # 静态资源文件
│  ├─ components          # 全局组件
│  ├─ config              # 全局配置
│  │  ├─ env.js           # 运行环境变量（重要）
│  ├─ constants           # 全局常量
│  ├─ helpful             # 一些封装 taro API 方法（行内调用）
│  ├─ hooks               # 自定义 hooks & 一些封装 taro API 方法（生命周期调用）
│  ├─ pages               # 项目所有页面
│  ├─ plugins             # 一些插件
│  ├─ router              # 路由跳转相关方法
│  ├─ stores              # pinia 状态中心管理
│  ├─ utils               # 工具库
│  ├─ app.config.js       # 小程序全局配置
│  ├─ app.js              # 入口文件
│  ├─ app.less            # 通用样式
│  ├─ index.html          # h5 入口文件
├─ .eslintrc.js           # eslint 校验配置
├─ .gitignore             # git 提交忽略
├─ .prettierignore        # 忽略 prettier 格式化
├─ .prettierrc.js         # prettier 配置
├─ project.config.json    # 微信小程序项目配置
├─ package.json           # 依赖包管理
└─ README.md              # README 介绍
```
